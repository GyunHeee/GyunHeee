## 이견희 | Software developer

새로운 문제를 찾고 문제 해결 과정을 즐기는 백엔드 개발자 이견희입니다. 
* 개발스펙에 한계가 있다고 생각이 되어도 ‘문제’를 해결하기 위해 다방면으로 고려하는 것을 좋아합니다.
* 기획자, 디자이너, 백엔드 개발자와의 의사소통을 즐기며, 상대방의 입장에서 대안을 제시하기 위해 노력합니다.
* 비즈니스 성장을 위한 새로운 기술을 배우고 도입하는 것의 중요성을 잘 알고 있습니다.
